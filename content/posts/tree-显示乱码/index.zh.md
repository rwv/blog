当使用 `tree` 命令来获取目录结构的时候，会得到乱码。

<!--more-->

## 解决方案

当使用 `tree` 时得到结果

```
$ tree
.
├── �\213�\225�\226\207件
└── �\213�\225�\226\207件夹

1 directory, 1 file
```

使用 `tree -N`得到正确输出

```
$ tree -N
.
├── 测试文件
└── 测试文件夹

1 directory, 1 file
```
